# dignas70th

Official static birthday invitation website for Digna's 70th birthday.

## Pages

- `index.html` — front cover page
- `invitation.html` — official invitation page
- `css/styles.css` — design, layout, responsive styling, animations
- `js/front.js` — opening button transition
- `js/invitation.js` — official page loader, countdown, scroll effects

## GitHub Pages setup

Repository name:

```text
dignas70th
```

Recommended GitHub Pages settings:

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

Expected live URL if published under the same GitHub account used for the portfolio:

```text
https://ronjr1994.github.io/dignas70th/
```

## Upload checklist

1. Create a new GitHub repository named `dignas70th`.
2. Upload all files from this folder directly into the repository root.
3. Go to repository **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose **main** and **/root**.
6. Save.
7. Wait a few minutes for GitHub Pages to publish.

## Important

Do not upload the whole folder as a nested folder inside the repository.  
The repository root should directly contain:

```text
index.html
invitation.html
css/
js/
README.md
.nojekyll
```
