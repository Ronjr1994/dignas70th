# Digna 70th Birthday Invitation Website

This is the rebuilt interactive webpage based on the uploaded Figma Make design file.

Open `index.html` directly in a browser, or upload the whole folder to Netlify, Vercel, GitHub Pages, or any static hosting.

Included:
- responsive cover page
- opening transition
- cinematic hero slideshow
- floating particles
- countdown
- photo mosaic
- swipeable filmstrip
- lightbox gallery
- event details
- program of activities
- embedded Google map
