# Digna 70th Birthday Invitation

GitHub Pages-ready invitation website.

## Updates in this version

- Restored the Figma typography: Great Vibes for names/headings and Playfair Display for age/program category titles.
- Applied the same Figma-style typography to the program dropdown section.
- Added the newly uploaded video as the first item in the **Golden Memories** section.
- Hero section now uses only the 10 photos uploaded in the latest message.
- Hero images are saved with GitHub-safe filenames: `hero-01.jpg` through `hero-10.jpg`.
- The new opening video was optimized to 1080p with audio preserved for faster mobile loading.
- Videos remain responsive for iPhone, iPad, Android, and desktop.
- `.nojekyll` is included so GitHub Pages serves all assets correctly.
- Image and video filenames do not begin with underscores.

## Upload structure

Upload these files/folders to the root of your GitHub repository:

```txt
index.html
.nojekyll
assets/
figma-make-source/
README.md
```

## Note about autoplay audio

The video can play with audio after the guest taps **Open Invitation**. iPhone, Android, and desktop browsers can still block audio when no user interaction has happened first.
